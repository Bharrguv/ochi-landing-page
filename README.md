

# Ochi-Style Landing Page

A modern, interactive, and visually appealing landing page inspired by the design aesthetics of **Ochi Design**. Built with **React**, **Tailwind CSS**, and **Framer Motion**, featuring animated elements like eye-catching marquees, rotating eyeballs, featured project cards, and a sleek footer.

## ðŸ”¹ Installation

1. **Clone the repository:**

```bash
git clone https://github.com/yourusername/ochi-landing-page.git
```

2. **Navigate to the project folder:**

```bash
cd ochi-landing-page
```

3. **Install dependencies:**

```bash
npm install
```

4. **Start the development server:**

```bash
npm start
```

5. Open your browser at `http://localhost:3000` to see the landing page.
